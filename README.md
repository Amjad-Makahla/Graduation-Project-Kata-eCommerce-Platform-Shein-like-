- Favorites list

  - Currently selected product
- AI-powered sentiment analysis system:
  - Analyzes customer comments
  - Classifies emotions (Positive, Neutral, Negative)
  - Assigns a sentiment score to each product

## Purpose
To enhance product marketing and help users understand customer opinions through AI analysis.
